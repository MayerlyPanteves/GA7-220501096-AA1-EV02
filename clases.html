Convenciones de Nomenclatura

Nomenclatura de Clases

Usar nombres descriptivos en inglés en formato kebab-case (todo en minúsculas separado por guiones).

<div class="user-profile-card"></div>

Nomenclatura de Métodos

Usar camelCase con verbos que describan la acción.

function calculateTotalPrice() {
    // Código del método
}

Convención para Comentarios

Comentarios claros y concisos. En CSS usar formato de bloque, en JS línea o bloque según extensión.

/*
 * Este es un comentario de bloque en CSS
 * Describe la sección o componente
 */
.navbar {
    /* Propiedades CSS */
}
// Comentario de línea en JavaScript
function init() {
    // Inicializa la aplicación
}
